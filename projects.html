Joe Dickinson Portfolio

I was approached by a tattoo artist to design and build a website to showcase their work. My first aim of this project was to ensure that the artist's work spoke for itself.

The key elements here were to let visitors see the tattoos and artwork as easily and possible and, secondly, make it easy to contact the tattoo artist.

It was built using HTML5, CSS, Sass and JavaScript.

Run Dem Crew

My aim for this project was to redesign an already existing website. In this case it was for the Run Dem Crew, a running club based in London. I felt that the original design was extremely cluttered so my initial idea was strip it down and give it a minimal design.

The club has a lot of great imagery that they use in their social media outlets which really helped this rebuild.

It features an image carousel, a page transition effect and a fixed navigation all built with JavaScript. It is also fully responsive.

The biggest challenge I faced with this project was creating the image carousel for the Gallery page. It was the first time I had used a jQuery plugin so getting used to implementing one was the biggest hurdle for me. After coming across a tutorial video on YouTube I was soon comfortable getting the plugin working exactly the way I wanted.

JavaScript Image Gallery

This image gallery was created with JavaScript and CSS. It works by adding and removing a "show" class to an image which is triggered by a click event.

The fade effect is created by a CSS transition attribute.

It was built using HTML5, CSS, Sass and JavaScript.

Family Christmas

For Christmas 2015 I decided to build a special and unique present for my family to enjoy. My thought process behind this project was to make it similar to an advent calendar. When ever the user hovers or focuses, as in on a touch screen device, a family photography is revealed.

It was built using HTML5, CSS, Sass and the transition effect is created through JavaScript. CSS is used to reveal the photos. From the initial design to final build this project took me just over a day to complete. I would of liked to of added more features to the page had I had more time.
